# 🌌 Parallel Lives Premium UI

An interactive web experience that lets users explore different career paths like Artist, Corporate, Entrepreneur, Traveler, and Scientist.

## 🚀 Features
- ✨ Glassmorphism UI with glowing effects
- 🎬 Smooth animations using GSAP
- 📊 Dynamic graph visualization (Canvas)
- 🎯 Career insights (Journey, Salary, Growth)
- 💡 Interactive side panel with stats

## 🛠️ Tech Stack
- HTML5
- CSS3 (Glassmorphism + Animations)
- JavaScript (Vanilla JS)
- GSAP (Animation Library)

## 🎯 Learning Outcome
- DOM manipulation
- UI/UX design principles
- Animation using GSAP
- Data visualization basics

## 🚀 How to Run
1. Clone the repository
2. Open `index.html` in browser
